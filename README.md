# duckshop

Duck Shop is a "desk-sized" vending machine with 4 slots. Instead of requiring payment, it can dispense items manually or trigger rewards automatically after completing goals like studying, reading, or finishing tasks.

#### why i did it 

Since childood ive always loved vending machines and dreaming about having one. Duck Shop is something useful but fun too.
well and this is my most ambitious project.

## stuff it haves/does

- 4 spiral vending slots
- dispenses items on demand
- can give rewards automatically
- ESP32 controlled
- compact desktop size
- modular design for future upgrades

Examples:
- press button: get snack
- study for 1 hour: reward
